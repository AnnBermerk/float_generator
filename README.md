# Генератор индивидуальных заданий: "Машинное представление вещественных чисел"

##  Описание

Приложение на языке C автоматически генерирует индивидуальные варианты учебных заданий и решения в виде подробных таблиц по теме **"Машинное представление вещественных чисел"**.

Для каждого варианта программа:
1) генерирует `k` случайных вещественных чисел в диапазоне `[a, b]` с `p` знаками после запятой;  
2) получает их «машинное представление» (в виде HEX-строки байтов числа);  
3) выполняет обратное преобразование из HEX обратно в число;  
4) вычисляет абсолютную погрешность и сохраняет результаты в Markdown-таблицы.


##  Структура проекта

### Входные данные (`input.txt`)

Файл `input.txt` должен лежать рядом с исполняемым файлом/исходниками и содержать 6 значений:

1. `n` — количество вариантов  
2. `k` — количество заданий (чисел) в варианте  
3. `bits` — разрядность: `32` или `64`
4. `a` — левая граница диапазона  
5. `b` — правая граница диапазона  
6. `p` — количество знаков после запятой

Формат чтения: `n k bits a b p` (разделители — пробелы/переводы строк).

**Пример `input.txt`:**
```txt
3 5 32 -10 10 2
```

###  Формат выходных файлов

После запуска программа создаёт:
```
├── Задания/                 # Каталог с заданиями
│   ├── variant_1.md
│   ├── variant_2.md
│   └── ...
└── Проверка/                # Каталог с решениями
    ├── variant_1.md
    ├── variant_2.md
    └── ...
```

#### Задание (`Задания/variant_n.md`):
```markdown
# Вариант 1

**Параметры:** разрядность=32, диапазон=[-10.00; 10.00], кол-во знаков после запятой=2, чисел=5

| № | Вещественное число |
|--:|-------------------:|
| 1 | -7.58 |
| 2 | -9.24 |
| 3 | 1.23 |
| 4 | -7.63 |
| 5 | 2.03 |

```

#### Решение (`Проверка/variant_n.md`):
```markdown
# Вариант 1 — проверка

**Параметры:** разрядность=32, диапазон=[-10.00; 10.00], кол-во знаков после запятой=2, чисел=5

| № | Вещественное число | Машинное представление | Точность |
|--:|-------------------:|:----------------------:|---------:|
| 1 | -7.58 | `C0F28F5C` | 7.629395e-08 |
| 2 | -9.24 | `C113D70A` | 2.288818e-07 |
| 3 | 1.23 | `3F9D70A4` | 1.907349e-08 |
| 4 | -7.63 | `C0F428F6` | 1.144409e-07 |
| 5 | 2.03 | `4001EB85` | 2.861023e-08 |

```
